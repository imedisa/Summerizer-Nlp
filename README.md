# سامانه خلاصه‌سازی متن فارسی

این پروژه یک **سامانه خلاصه‌سازی متن فارسی** است که از سه رویکرد
**استخراجی (Extractive)**، **مولد (Abstractive)** و **هیبرید (Hybrid)**
پشتیبانی می‌کند.
هدف پروژه، تولید خلاصه‌ای کوتاه، معنادار و قابل تنظیم از متون فارسی است.
سامانه به‌صورت API پیاده‌سازی شده و قابل استفاده در سرویس‌های بک‌اند می‌باشد.

## معرفی پروژه

در این سامانه از ترکیب روش‌های کلاسیک پردازش زبان طبیعی و مدل‌های مدرن یادگیری
عمیق استفاده شده است، از جمله:

- الگوریتم‌های مبتنی بر گراف مانند **TextRank**
- مدل‌های مبتنی بر **Transformer** برای خلاصه‌سازی مولد
- ترکیب دو روش فوق در قالب یک روش **هیبرید**

پروژه برای کار با متون فارسی و متون طولانی طراحی شده است.



## قابلیت‌ها

- خلاصه‌سازی استخراجی با استفاده از TextRank
- خلاصه‌سازی مولد با مدل فارسی mT5
- خلاصه‌سازی هیبرید (Extractive → Abstractive)
- پشتیبانی از متون طولانی با استفاده از chunking
- امکان کنترل طول خلاصه
- پیش‌پردازش اختصاصی زبان فارسی
- خروجی همراه با متادیتا (امتیاز جملات، اندیس‌ها)


## روش‌ها

### خلاصه‌سازی استخراجی (Extractive)

در این روش، جملات مهم متن با استفاده از مراحل زیر انتخاب می‌شوند:

- محاسبه TF-IDF جملات
- محاسبه شباهت جملات با Cosine Similarity
- ساخت گراف شباهت
- محاسبه PageRank برای امتیازدهی جملات
- انتخاب جملات برتر با حفظ ترتیب اصلی متن

<img width="853" height="882" alt="image" src="https://github.com/user-attachments/assets/8252bea1-7113-4d2d-84e6-e6c07798ab4a" />


### خلاصه‌سازی مولد (Abstractive)

این روش از مدل‌های Transformer استفاده می‌کند و شامل مراحل زیر است:

- پشتیبانی از GPU در صورت وجود
- تقسیم متن به چانک‌های هم‌پوشان
- خلاصه‌سازی هر چانک به‌صورت جداگانه
- تولید خلاصه نهایی در مرحله دوم
<img width="1049" height="880" alt="image" src="https://github.com/user-attachments/assets/95959eb8-f476-4b10-8f5c-2625f8a766cd" />



### خلاصه‌سازی هیبرید (Hybrid)

در روش هیبرید، ابتدا خلاصه‌سازی استخراجی انجام می‌شود تا بخش‌های مهم متن
شناسایی شوند. سپس خروجی این مرحله به مدل مولد داده می‌شود تا یک خلاصه
نهایی منسجم‌تر و کوتاه‌تر تولید شود.

این روش برای متون طولانی یا متون دارای نویز عملکرد بهتری دارد.

```bash
docker compose up
```
