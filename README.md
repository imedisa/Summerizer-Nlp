# Persian Text Summarization System

سامانه خلاصه‌سازی متن فارسی با پشتیبانی از دو رویکرد **استخراجی (Extractive)** و **مولد (Abstractive)**  
این پروژه برای تولید خلاصه‌ای کوتاه، معنادار و قابل تنظیم از متون فارسی طراحی شده است.



## معرفی پروژه
این سامانه با هدف خلاصه‌سازی متون فارسی طراحی شده و امکان استفاده از:
- الگوریتم‌های کلاسیک مبتنی بر گراف (TextRank)
- مدل‌های Transformer برای خلاصه‌سازی مولد  
را فراهم می‌کند.

پروژه به‌صورت API پیاده‌سازی شده و قابل استفاده در سرویس‌های بک‌اند است.


## قابلیت‌ها
- خلاصه‌سازی استخراجی با TextRank
- خلاصه‌سازی مولد با مدل mT5 فارسی
- پشتیبانی از متون طولانی با chunking
- کنترل طول خلاصه
- پیش‌پردازش اختصاصی زبان فارسی
- خروجی همراه با متادیتا (امتیاز جملات، اندیس‌ها)



## روش‌ها

### Extractive Summarization
- مبتنی بر TF-IDF و Cosine Similarity
- ساخت گراف شباهت جملات
- محاسبه PageRank
- انتخاب جملات مهم با حفظ ترتیب اصلی

### Abstractive Summarization
- استفاده از مدل Transformer
- پشتیبانی از GPU
- تقسیم متن به چانک‌های هم‌پوشان
- خلاصه‌سازی دو مرحله‌ای (Chunk-level → Final)
